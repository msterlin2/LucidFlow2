# Lucid Flow 🧠✨

**Lucid Flow** is a premium, high-performance cognitive training application designed to sharpen focus, memory, and logical reasoning. Built with a modern, fluid interface, it provides a gamified experience to track and improve your mental agility.

## 🚀 Features

### 🎮 Cognitive Games
- **Stroop Challenge:** Test your selective attention and processing speed by identifying ink colors while ignoring conflicting word meanings.
- **Word Unscramble:** Enhance your linguistic processing and vocabulary retrieval by unscrambling jumbled letters.
- **Memory Match:** Sharpen your spatial memory by finding pairs in an increasingly complex grid.
- **Equation Master:** Solve visual logic puzzles to boost your mathematical reasoning and mental calculation speed.

### 📊 Neural Insights
- **Cognitive Profile:** Track your performance across Focus, Memory, Logic, and Linguistic categories.
- **Progress Tracking:** Monitor your Peak Focus, Total Training Time, and Session counts.
- **Consistency Goals:** Stay motivated with a 7-day streak system and weekly targets.

### 👤 Personalization
- **Custom Profiles:** Set your username and upload a custom avatar.
- **Adaptive Leveling:** Progress through ranks from "Novice Thinker" to "Infinite Mind" based on your performance.

### 📱 Mobile Ready (PWA)
- **Installable:** Add Lucid Flow to your home screen on iOS or Android.
- **Full Screen:** Enjoy a native-like experience without browser chrome.
- **Responsive:** Optimized for all screen sizes, from mobile to desktop.

## 🏁 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/lucid-flow.git
   ```
2. Navigate to the project directory:
   ```bash
   cd lucid-flow
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

### Build
Create a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` directory.

## 🌐 Deployment

### Google Cloud Run

### GitHub Pages / Vercel / Netlify
